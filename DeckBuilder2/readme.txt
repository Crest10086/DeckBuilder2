【图形组卡器使用说明】

本程序需要.NET Framework 2.0支持，运行出错的朋友请先安装.NET Framework 2.0。
下载地址：http://www.newhua.com/soft/38669.htm

1、第一次运行前，请先准备好卡图。没有卡图朋友，本组卡器使用工作室专用卡图，即与NBX、天堂狐查卡器、迷你卡片等程序使用的卡图完全一致，已经有的朋友无需重复下载，没有的可以到http://www.ocgsoft.cn/download.aspx处下载。不需要显示卡图的人，请跳过这一步。

2、准备好卡图后，请执行ConveterTools.exe里的生成略缩图功能。执行该功能需要较长时间，窗口会假死，请耐心等待。不需要显示略缩图的人，请跳过这一步。

3、第一次执行组卡器，请先运行设置功能，指定好NBX目录。卡图和卡组目录建议使用NBX的默认目录。略缩图目录通常无需改动。设定完路径请重新启动。


【更新数据】
    如果天堂狐查卡器或者迷你卡片更新了卡片数据，而我没有及时更新的话，你们可以使用ConveterTools自行将新的数据导入到本组卡器中。
    使用YGORPO的朋友，需要在导入完狐查数据之后，再导入YGOPRO补充数据，否则载入YGOPRO卡组时可能出错，且无法使用效果分类搜索。
    有新卡图的时候，请再次运行生成略缩图功能，可以对新的卡图生成略缩图，而已经有的略缩图不会被重复生成。


【使用DIY卡】
    1、按照NBX的说明，在NBX中添加DIY卡
    2、运行ConveterTools，点击 导入数据--DIY卡，选择diycards.dll
    3、点击生成略缩图，选择diyimg目录
    4、运行deckbuilder2.exe，在设置里勾上允许DIY卡


【卡片词霸使用说明】
1、卡片词霸启动时默认会最小化到系统托盘。
2、取词方式可以在设置窗口里选择。默认为鼠标直接取词。
3、快捷键F11打开/关闭取词功能，F10呼出主窗口
4、如果不能取词，可能是被杀毒软件或者防护软件拦截掉了，解决方案请去搜索金山词霸与你使用的安全软件有兼容性问题如何处理。
5、如果仅仅在网页里不能取词，谷歌浏览器和Firefox2因为使用了特别的渲染引擎，当前的取词模块不能支持。FF2用户请升级到FF3。如果是IE或者其他IE内核浏览器，请参考上一条处理。
